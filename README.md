An offline practice tool for challenge running in Sekiro

This tool is strictly for offline practice only, it directly manipulates game memory which violates Terms of Service and will most likely lead to a ban if you use this online.

Includes:
* Saving / Restoring position
* Character cheats such as No Death/No Damage and more
* Add Experience & Sen
* Change New Game Cycle
* Infinite Confetti / Gachiin
* Idol / Boss warps
* Unlock All Idols
* Disable Ai / Enemies no Death and more
* Skip Dragon Phase 1
* Force Dragon Final Attack
* Force a specific starting combo in Dragon Phase 2 (Useful for speedruns)
* Butterfly no Summons
* Many options for the locked on target
* Hitbox View, Event view, Sound view
* Adjust Game Speed
* Noclip
* Free cam
* Open various menus and shops from anywhere
* Spawn items / prosthetics
* Learn / Unlearn skills
* Toggle events such as Demon Bell/No Kuro's charm/daytime scaling and more
* Disable Events
* Set/Get specific events by their ids
* No tutorials
* No Logo
* Toggle being able to save in combat 
* Disabling menu music / Lower default menu sound
* Hotkeys for most things


Usage:
Unzip the download and run the exe, it does not matter if you open the app before the game or after.
